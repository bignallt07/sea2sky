Voucher Redemption- Big Fun Run Series


Whether you purchased yourself a gift voucher or were given one as a gift, follow the instructions below to redeem.

Ensure you redeem before the voucher expires!


To Redeem

Click the link below for the event you wish to sign up for.
Codes are valid for specific events so ensure you use the correct one. Or if you have purchased multiple events use the Series link)

Big Easter Run
raceroster.com/events/2020/28368/the-big-easter-run

Big Elf Run
raceroster.com/events/2020/28356/the-big-elf-run

The Big Fun Run Series
raceroster.com/series/2020/28402/the-big-fun-run-series

Click on "Register" button

Login or create a Race Roster account

Select the run type you chose (1k, 5k, 10k, 15k)

Complete the Registration questions, once you reach the bottom of the page you will see a box for "Enter promo code or gift code" Enter your voucher code in here

You can then click continue, OR if you wish to add another participant: click Add another registrant

If you have codes for multiple events (Easter and Elf) please choose "Register for Series" on the main page or you may need to complete this process twice


Looking forward to seeing you at the Start Line!

If you have any issues with your code,
please email: registration@sea2skyevents.com
and quote your name, code and issue
